Fix three issues with character avatar display.

ISSUE 1 — AVATAR SIZE STANDARDIZATION
The reference standard is 박철수 과장's avatar appearance in the sidebar. All other characters must match this size and zoom level.

Currently:
- Min Choi: too zoomed in, face is too large and gets cut off
- James, Sarah, Lisa, User: too small/zoomed out compared to Park

Fix by using these exact CSS values for ALL circular avatars throughout the app (sidebar, commute screen, chat bubbles, chat header):

Container:
  width: 44px
  height: 44px
  border-radius: 50%
  overflow: hidden
  flex-shrink: 0

Image inside container:
  width: 100%
  height: 180%
  object-fit: cover
  object-position: center 8%
  image-rendering: pixelated
  display: block

The key values are height: 180% and object-position: center 8%.
This shows the upper portion of the character (face + shoulders) centered in the circle.
Apply these EXACT same values to every single avatar in the app — do not use different values for different characters. The same CSS must apply to all characters consistently.
