FIX 2 — AUTO BACKGROUND COLOR FROM CHARACTER IMAGE

For character profile cards and commute screen NPC cards, automatically extract the dominant color from each character's pixel art image and use a light tinted version as the card background.

Implementation using Canvas API (built into browser, no installation needed):

Create a utility function getAvatarBgColor(imageSrc: string): Promise<string>

function getAvatarBgColor(imageSrc) {
  return new Promise((resolve) => {
    const img = new Image()
    img.crossOrigin = 'anonymous'
    img.onload = () => {
      const canvas = document.createElement('canvas')
      canvas.width = img.width
      canvas.height = img.height
      const ctx = canvas.getContext('2d')
      ctx.drawImage(img, 0, 0)
      
      // Sample colors from the middle portion of the image (clothing area)
      const startY = Math.floor(img.height * 0.3)
      const endY = Math.floor(img.height * 0.6)
      const sampleX = Math.floor(img.width / 2)
      
      let r = 0, g = 0, b = 0, count = 0
      for (let y = startY; y < endY; y += 3) {
        const pixel = ctx.getImageData(sampleX, y, 1, 1).data
        // Skip transparent, white, and skin-tone pixels
        if (pixel[3] > 128 && !(pixel[0] > 220 && pixel[1] > 220 && pixel[2] > 220)) {
          r += pixel[0]; g += pixel[1]; b += pixel[2]; count++
        }
      }
      
      if (count > 0) {
        r = Math.floor(r / count)
        g = Math.floor(g / count)  
        b = Math.floor(b / count)
        // Create a very light tinted version (mix with white at 85%)
        const lr = Math.floor(r * 0.15 + 255 * 0.85)
        const lg = Math.floor(g * 0.15 + 255 * 0.85)
        const lb = Math.floor(b * 0.15 + 255 * 0.85)
        resolve(`rgb(${lr}, ${lg}, ${lb})`)
      } else {
        resolve('#F7F5F1') // fallback
      }
    }
    img.onerror = () => resolve('#F7F5F1')
    img.src = imageSrc
  })
}

Use this function in:
1. Character profile cards (/characters page): set card background to extracted color
2. Commute screen NPC status cards: set card background to extracted color
3. Left sidebar NPC hover popup: use extracted color as background

Call this function when the component mounts using useEffect:
useEffect(() => {
  getAvatarBgColor('/characters/sarah.png').then(color => setSarahBg(color))
  getAvatarBgColor('/characters/james.png').then(color => setJamesBg(color))
  // etc for each character
}, [])

This way when character images are updated, the background colors automatically update to match without any manual changes needed.
