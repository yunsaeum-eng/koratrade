KoraTrade — English UI Overhaul
================================
Read ALL source files in the project first, then implement everything below.
This applies ONLY when ui_language === 'english'. Korean mode must remain unchanged.

---

CONTEXT
-------
When a user selects English during onboarding, the entire play UI must be in English.
Currently most UI text is hardcoded in Korean. We need to make everything language-aware.
The approach: add English fields to all data files, and use ui_language to switch.

TASK 2 — useLanguage hook
---------------------------
Update src/hooks/useLanguage.ts (or create it if missing):

export function useLanguage() {
  const { profile } = useAuth()
  const lang = profile?.uiLanguage === 'english' ? 'english' : 'korean'
  const isEn = lang === 'english'
  return { lang, isEn }
}

This hook is used everywhere to decide which text to show.
