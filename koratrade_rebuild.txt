Delete all existing code and rebuild KoraTrade from scratch. The entire NPC system must be powered by Claude API in real time. No pre-scripted NPC dialogue at all.

CRITICAL ARCHITECTURE RULE:
Every single NPC message must go through Claude API. There are zero hard-coded NPC dialogue strings anywhere in the codebase. The only things that are fixed are: episode context, character personalities, and learning objectives. Everything NPCs say is generated in real time by Claude API based on what the user actually said.

---

TECH STACK:
- Next.js 14 with TypeScript
- Tailwind CSS
- localStorage for all data persistence (no database needed)
- Claude API (claude-haiku-4-5) for ALL NPC responses
- API key from .env.local: ANTHROPIC_API_KEY

STEP 1: CREATE THE API ROUTE

Create /app/api/chat/route.ts

This endpoint receives:
- npcId: which NPC is responding (james, sarah, lisa, min, park, klaus)
- userMessage: what the user just said
- conversationHistory: last 15 messages in the conversation
- episodeId: current episode (s1ep01, s1ep02 etc)
- language: korean or english
- relationshipLevel: 0-100 number
- userProfile: object with name, goal, englishLevel, industryInterest

It builds a system prompt based on npcId and episodeId, calls Claude API, and streams the response back.

STEP 4: ETIQUETTE FEEDBACK SYSTEM

After generating each NPC response, also call Claude API with a separate quick analysis:

Prompt: "The user sent this message: [USER_MESSAGE] to [NPC_NAME] who is their [RELATIONSHIP_TYPE] in a Korean workplace. The current situation is [SITUATION]. Is there any etiquette issue? Check for: using informal speech with seniors, being too blunt, ignoring a question, one-word replies to detailed questions, unprofessional email language. If there is an issue, respond with JSON: {hasIssue: true, severity: mild or serious, tip: one sentence feedback in Korean}. If no issue: {hasIssue: false}"

If hasIssue is true:
- mild severity: have the NPC react slightly differently (cooler tone, gentle correction in their response)
- serious severity: show a small toast popup at bottom of screen with the tip text, styled with 💼 icon and gold border
